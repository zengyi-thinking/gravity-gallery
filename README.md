## 🎮 在线玩

**访问地址：** https://zengyi-thinking.github.io/gravity-gallery/

## 🕹️ 操作说明

| 操作 | 说明 |
|------|------|
| **点击画布** | 放置一颗星球 |
| **按住拖拽** | 发射流星（拖拽方向=飞行方向） |
| **「清除全部」** | 清空所有天体 |
| **「生成演示」** | 一键生成五星连珠演示 |
| **「彗尾轨迹」** | 开关流星拖尾效果 |

## 🎯 游戏技巧

- 多放几颗星球，看它们相互引力下的混沌运动
- 用流星去"撞击"星球，观察合并效果
- 拖拽距离越远，流星速度越快

## 🦞 技术栈

- Pure HTML5 Canvas + Vanilla JS
- 万有引力物理模拟
- 单文件，无依赖，零成本部署
